Este site foi desenvolvido como parte da disciplina de Programação Web no Instituto Federal do Piauí (IFPI), utilizando o framework Django. O objetivo deste projeto é servir como um repositório para as atividades e projetos desenvolvidos ao longo do curso, permitindo uma fácil visualização e consulta do progresso.

Cada atividade presente neste site reflete o aprendizado e as habilidades adquiridas durante as aulas, desde conceitos básicos até a implementação de funcionalidades mais avançadas. Este espaço não apenas organiza o trabalho desenvolvido, mas também oferece um local centralizado para revisões, melhorias e futuras referências.

Espero que este projeto sirva como uma ferramenta útil tanto para meu próprio desenvolvimento quanto para outros que estejam interessados em explorar as aplicações práticas do Django no contexto da programação web.
